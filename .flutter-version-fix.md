# Flutter SDK Issue Fix

If you encounter the `_LineCaretMetrics` exhaustive pattern matching error:

1. This is a known issue in Flutter 3.38.7
2. Try switching Flutter channels:
   ```bash
   flutter channel beta
   flutter upgrade
   ```

3. Or use FVM to pin a working version:
   ```bash
   fvm install 3.24.0
   fvm use 3.24.0
   ```

4. The error is in Flutter's SDK, not your code
